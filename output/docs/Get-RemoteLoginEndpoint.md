# Get-RemoteLoginEndpoint

## SYNOPSIS
Gets a remote login endpoint to a VM that is hosting a multiplayer server build.

## SYNTAX

```
Get-RemoteLoginEndpoint -BuildId <String> -Region <String> -VMId <String> [-CustomTags <IAny>] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Gets a remote login endpoint to a VM that is hosting a multiplayer server build.
